# python-web-scrapper

# 개요 
- 파이썬,beautifulsoup와 라이브러리를 활용하여 특정 사이트에 있는 정보들을 수집하여 DataBase에 주기적으로 저장한다. 그리고 사용자가 검색하는 검색어를 처리하여 DataBase 저장되어있는 정보를 검색어와 비교하여 사용자에게 정보를 보여준다.
